# VCd03 — B2B LeadFlow Agent 2026

> Краткая спецификация этапа 0. Полное пользовательское ТЗ предоставлено в рамках курса VCd03.

## Назначение

Агент для B2B lead generation: поиск компаний по нише и региону, формирование воронки писем, ручное подтверждение отправки, отслеживание ответов и заинтересованных лидов.

## Зафиксированные ограничения MVP

| Ограничение | Значение |
|---|---|
| Роли | Одна роль администратора |
| Кампания | 1 ниша + 1 регион |
| Компании | Максимум 30 |
| Письма на адресат | Максимум 3 |
| Подтверждение | Ручное, обязательное |
| Холодная рассылка | Запрещена на этапе 0 |
| Email-провайдер по умолчанию | `TestEmailProvider` |
| Источник компаний (этап 0) | `TestSourceAdapter` |
| LLM | Не может самостоятельно запускать отправку |
| Секреты | Не хранятся в репозитории и не попадают в логи |
| Аварийная остановка | `SYSTEM_STOP_ALL=true` блокирует исходящие задачи |

## Этап 0 — scope

- Docker Compose каркас (frontend, backend, worker, scheduler, postgres, redis)
- Health / readiness / version API
- Dashboard с индикаторами состояния
- Тестовые провайдеры без реальной отправки и поиска
- Celery ping-задача, пустой beat schedule

## Следующие этапы (не в scope этапа 0)

- Модели кампаний, компаний, очереди
- SMTP/IMAP провайдеры (после явного разрешения)
- Реальный поиск компаний
- Авторизация администратора
- Деплой на VPS
